# DeployWizard

Simple SSH deployment tool with wizard-driven interface for deploying websites to hosting servers

## Quick Start

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

The frontend will be available at http://localhost:5173
The backend API will be available at http://localhost:3001

## Project Structure

```
├── frontend/          # React + Vite frontend
│   ├── src/

│   └── ...
├── backend/           # Express.js backend

│   ├── database.js   # SQLite database
│   └── server.js     # Express server
└── scripts/          # Helper scripts
```

## Features

SSH connection wizard with secure credential management, drag-and-drop folder selection and file browser, automated deployment pipeline with pre-deployment validation, real-time deployment progress tracking with detailed logs, server management with multiple hosting provider support, deployment history and rollback capabilities, file comparison and sync detection, automated backup creation before deployment, deployment scheduling and automation, team collaboration with shared server configurations.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Tailwind CSS
